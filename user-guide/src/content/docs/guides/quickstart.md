# クイックスタート

x402で初めてコンテンツを購入するまでの流れを説明します。

## 全体の流れ

```
① アカウント登録 → ② パスキー設定 → ③ 商品を選ぶ → ④ 購入する → ⑤ コンテンツを楽しむ
```

## Step 1: アカウント登録

1. トップページの「ログイン」ボタンをクリック
2. 「新規登録」タブを選択
3. メールアドレスとお名前を入力
4. 「登録」ボタンをクリック

:::note
メールアドレスは確認メールの送信には使用されません。ログイン時の識別に使用されます。
:::

## Step 2: パスキー設定

1. 登録完了後、パスキー設定画面が表示されます
2. 「パスキーを登録」ボタンをクリック
3. デバイスの認証（指紋/顔認証）を行う
4. 登録完了！

:::tip
パスキーを登録すると、次回から指紋認証だけでログインできます。
:::

## Step 3: 商品を選ぶ

1. トップページで商品一覧を確認
2. 興味のある商品カードをクリック
3. 商品詳細ページで内容を確認

## Step 4: 購入する

1. 商品詳細ページの「購入する」ボタンをクリック
2. 決済確認画面で金額を確認
3. 「支払いを確定」をクリック
4. 処理完了を待つ（数秒）

## Step 5: コンテンツを楽しむ

購入完了後、すぐにコンテンツにアクセスできます！

- 記事：その場で全文が表示されます
- 画像：高解像度版をダウンロード
- 音楽：再生プレイヤーが表示されます

## 購入履歴の確認

ヘッダーの「購入履歴」から、過去の購入を確認できます。トランザクションの詳細も確認可能です。
