# x402決済プラットフォームとは

x402は、デジタルコンテンツをかんたんに購入できる決済プラットフォームです。

## 特徴

### シンプルな購入体験

ブロックチェーンの複雑な操作は一切不要。通常のオンラインショッピングと同じ感覚でデジタルコンテンツを購入できます。

### 安全なパスキー認証

パスワードを覚える必要はありません。お使いのデバイスの指紋認証やFace IDでログインできます。

### 即時アクセス

購入完了と同時に、コンテンツへのアクセス権が付与されます。待ち時間なしでコンテンツを楽しめます。

## 購入できるコンテンツ

| 種類 | 例 |
|------|-----|
| 記事 | プレミアム記事、レポート |
| 画像 | デジタルアート、写真素材 |
| 音楽 | BGM、楽曲 |
| 動画 | チュートリアル、映像作品 |

## 決済について

x402では、Avalancheブロックチェーン上のAVAXトークンで決済が行われます。ただし、ユーザーが直接ブロックチェーンを操作する必要はありません。プラットフォームが代行して処理を行います。

:::tip[ポイント]
ブロックチェーンの知識がなくても、安心してご利用いただけます。
:::
