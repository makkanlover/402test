# パスキー認証

パスキー（Passkey）は、パスワードに代わる新しい認証方式です。指紋認証や顔認証を使って、安全かつ簡単にログインできます。

## パスキーとは

### 従来のパスワード認証との違い

| 項目 | パスワード | パスキー |
|------|-----------|---------|
| 覚える必要 | あり | なし |
| 入力の手間 | 毎回入力 | 指紋/顔認証のみ |
| フィッシング耐性 | 弱い | 強い |
| 使い回しリスク | あり | なし |

### 対応デバイス

パスキーは以下のデバイス/ブラウザでご利用いただけます：

- **Windows**: Windows Hello（指紋/顔/PIN）
- **macOS**: Touch ID / Face ID
- **iOS/iPadOS**: Face ID / Touch ID
- **Android**: 指紋認証 / 画面ロック

## パスキーの登録

### Step 1: パスキー登録画面を開く

![パスキー登録ボタン](/screenshots/passkey-register-button.png)

ログイン後、アカウント設定または認証画面から「パスキーを登録」をクリックします。

### Step 2: デバイスの認証

![デバイス認証ダイアログ](/screenshots/passkey-dialog.png)

ブラウザから認証ダイアログが表示されます。お使いのデバイスに応じて：

- **Windows**: Windows Helloで認証
- **Mac**: Touch IDで指紋認証
- **iPhone**: Face IDまたはTouch ID
- **Android**: 指紋認証または画面ロック

### Step 3: 登録完了

![パスキー登録完了](/screenshots/passkey-registered.png)

認証が成功すると、パスキーが登録されます。次回からこのパスキーでログインできます。

## パスキーでログイン

### Step 1: ログイン画面を開く

1. 「ログイン」ボタンをクリック
2. メールアドレスを入力
3. 「パスキーでログイン」をクリック

### Step 2: デバイスで認証

ブラウザの認証ダイアログが表示されたら、指紋認証または顔認証を行います。

### Step 3: ログイン完了

認証が成功すると、自動的にログインが完了します。

## よくある質問

### Q: パスキーを複数のデバイスで使える？

A: はい。同じアカウントに複数のパスキーを登録できます。スマートフォンとPCなど、複数のデバイスにそれぞれパスキーを登録しておくと便利です。

### Q: パスキーを紛失したら？

A: デバイスを紛失した場合でも、別のデバイスに登録されたパスキーでログインできます。すべてのデバイスを紛失した場合は、メールアドレスでの本人確認が必要になる場合があります。

### Q: パスキーに対応していないブラウザの場合は？

A: パスキー非対応のブラウザでは、パスキー認証機能は利用できません。対応ブラウザ（Chrome、Safari、Firefox、Edgeの最新版）をご使用ください。

### Q: 他人に指紋を登録されるリスクは？

A: パスキーはデバイスに紐づいています。あなたのデバイスであなたの指紋/顔でのみ認証できます。第三者がパスキーを登録・使用することはできません。
