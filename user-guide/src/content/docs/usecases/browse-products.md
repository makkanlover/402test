# 商品を閲覧する

x402で販売されている商品を探す方法を説明します。

## トップページ（商品一覧）

![商品一覧画面](/screenshots/product-list.png)

トップページにアクセスすると、販売中の商品が一覧表示されます。

### 商品カードの見方

各商品カードには以下の情報が表示されます：

| 項目 | 説明 |
|------|------|
| サムネイル | 商品のプレビュー画像 |
| 商品名 | コンテンツのタイトル |
| 種類 | 記事/画像/音楽/動画 のいずれか |
| 価格 | AVAX建ての価格 |

### 商品の種類

商品カードにはタイプを示すラベルが付いています：

- 📝 **記事** - テキストコンテンツ
- 🖼️ **画像** - デジタルアート、写真
- 🎵 **音楽** - BGM、楽曲ファイル
- 🎬 **動画** - 映像コンテンツ

## 商品詳細ページ

![商品詳細画面](/screenshots/product-detail.png)

商品カードをクリックすると、詳細ページが表示されます。

### 詳細ページの内容

1. **大きなサムネイル** - より詳しくプレビューを確認
2. **商品説明** - コンテンツの詳細な説明
3. **価格表示** - 支払い金額
4. **購入ボタン** - クリックで購入フローへ

### 購入前のプレビュー

未購入の商品は、一部のプレビューのみ表示されます。全体を見るには購入が必要です。

:::note[プレビューについて]
- 記事：冒頭の一部のみ表示
- 画像：低解像度またはウォーターマーク付き
- 音楽：サンプル（30秒程度）
- 動画：トレーラーまたは冒頭のみ
:::

## 購入済み商品の表示

すでに購入した商品の詳細ページでは、「購入済み」と表示され、フルコンテンツにアクセスできます。
