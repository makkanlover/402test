# 購入履歴を確認する

過去の購入を確認し、トランザクションの詳細を見る方法を説明します。

## 購入履歴ページへのアクセス

![ナビゲーション](/screenshots/nav-history.png)

ヘッダーの「購入履歴」リンクをクリックすると、購入履歴ページが表示されます。

:::note
購入履歴を見るにはログインが必要です。
:::

## 購入履歴一覧

![購入履歴画面](/screenshots/history-list.png)

購入履歴ページでは、過去のすべての購入が一覧表示されます。

### 表示される情報

| 項目 | 説明 |
|------|------|
| 商品名 | 購入した商品のタイトル |
| 金額 | 支払った金額 |
| ステータス | completed / pending / failed |
| 購入日時 | 購入した日時 |
| トランザクション | ブロックチェーン上のTxハッシュ |

### ステータスの意味

- ✅ **completed** - 決済完了、コンテンツにアクセス可能
- ⏳ **pending** - 処理中
- ❌ **failed** - 決済失敗

## トランザクションの確認

![トランザクション詳細](/screenshots/tx-detail.png)

各購入履歴の「トランザクションを確認」リンクをクリックすると、Snowtrace（Avalanche Fuji Testnetのエクスプローラー）で詳細を確認できます。

### Snowtraceで確認できる情報

- **From**: 送金元アドレス（プラットフォーム）
- **To**: 送金先アドレス（販売者）
- **Value**: 送金額（AVAX）
- **Block**: トランザクションが含まれるブロック番号
- **Status**: Success / Failed

## 購入した商品へのアクセス

購入履歴から商品名をクリックすると、その商品の詳細ページに移動します。購入済みなので、フルコンテンツにアクセスできます。

## よくある質問

### Q: 購入したのに履歴に表示されない

A: ページをリロードしてみてください。それでも表示されない場合は、決済処理中に問題が発生した可能性があります。

### Q: ステータスが「pending」のまま

A: 通常は数秒で完了しますが、ブロックチェーンの混雑状況により時間がかかる場合があります。数分待ってからリロードしてください。

### Q: ステータスが「failed」になった

A: 決済が失敗しています。商品詳細ページから再度購入をお試しください。同じ商品を購入済みでなければ、再購入が可能です。
