## 撮影のポイント

1. **解像度**: 1600x900以上を推奨
2. **ブラウザ**: Chrome/Edgeのデベロッパーツールでデバイスエミュレーションを使用
3. **ダークモード**: ライトモードで撮影
4. **個人情報**: テスト用のダミーデータを使用

## 画像形式

- PNG形式を推奨
- ファイルサイズが大きい場合はTinyPNGなどで圧縮
- 必要に応じてWebPに変換

## プレースホルダー

`placeholder.svg` は開発時のプレースホルダーです。本番前に実際のスクリーンショットに置き換えてください。
